# Credit Card Fraud Detection

## 📌 Project Overview
This project focuses on detecting fraudulent credit card transactions using Machine Learning techniques.

The dataset is highly imbalanced, with fraudulent transactions representing less than 1% of total transactions.

---

## 🚀 Models Used
- Logistic Regression
- Random Forest

---

## ⚙️ Techniques Applied
- Data preprocessing
- Handling imbalanced data (SMOTE)
- Model comparison
- Precision, Recall & F1-score evaluation

---

## 📊 Final Model Performance (Random Forest)

- Precision (Fraud): 0.86
- Recall (Fraud): 0.85
- F1-Score (Fraud): 0.86

---

## 💼 Business Impact

High recall ensures that most fraudulent transactions are detected, minimizing financial loss for banks.
